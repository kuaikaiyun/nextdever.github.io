在AndroidStudio中用style提升撸layout代码速度小技巧

  • Android Develop
Android Dever

用一个styles.xml文件干掉所有的 android:layout_width & android:layout_height,同时提高速度，压缩代码行数的小技巧。

–>使用styles_base.xml文件提升撸layout代码速度

  1. 在 res\values\ 新建一个xml文件：styles_base.xml 代码如下：

  2. 进入设置（ Ctrl+Alt+S || File->Settings） -> Editor -> General -> Smart Keys
    在右边面板 XML/HTML 分组设置下 找到 Insert required attributes on tag completion
    将该选项取消勾选(写layout时将不会自动生成android:layout_width & android:layout_height)

  3. 在xml中使用该style，eg: <TextView style="@style/base_ww" />
    下面给出了一个具体的示例，现在，原本三行的只需要一行就能搞定!!!

4.在写XML时，配合Android Studio的提示功能，比如你要写一个 <TextView/> 那么你只需要依次键入 < -> T -> Enter -> ←← -> Enter -> s(ty) -> Enter -> bww(base_ww) -> Enter


:: 注意，在RemoteView中是不能使用style的


用这种方式不仅仅压缩了代码，重要的是丝毫不会打断你 快速的思维 和 一套带走 de 组合技能。！！！
