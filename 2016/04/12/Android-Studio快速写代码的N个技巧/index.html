Android Studio快速写代码的N个技巧

  • Android Develop
Android Dever

总结我的日常开发中发现的各种技巧，基于 Keymap: Eclipse

  • 利用style快速撸 Layout xml

在我的另一篇blog中已经介绍了，主要是缩短部分代码，另外熟悉这种方式之后手速快，直接手跟着思路根本停不下来！

参考在AndroidStudio中用style提升撸layout代码速度小技巧


  • 不打开 strings.xml colors.xml dimens.xml 直接添加resources

姿势讲解：在撸xml时，在确实需要创建一个resourses时，接着撸，不要停！！！

比如：撸到需要一个color resources，而colors.xml中却没有，这时接着写：android:background="@color/bg_titleBar", @color/bg_titleBar会飘红，这时Alt+Enter,选择 Create color value resource \'bg_titleBar\' ，AS会弹出一个框，键入值回车即可

: : 注意 styles.xml, styles 可以先创建这个style(AS弹出输入参数值对话框时直接回车)，然后Ctrl+鼠标跟过去再把剩下的补全即可。

  • 在已打开的文件间切换（编辑区）

Ctrl + PageUp 切换到上一个文件

Ctrl + PageDown 切换到下一个文件

  • 重命名（包括文件名，变量名）

    Alt + shift + R

: : 改变文件名或者变量名时，同时AS会自动的改变所有的引用（所有出现过这个名字的地方）

  • Run & Debug

Alt + Shift + X 运行

Alt + Shift + D 调试模式运行
